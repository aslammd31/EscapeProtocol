# Escape Protocol

## Overview

Escape Protocol is a 2D survival game built using HTML, CSS, and JavaScript.

The player controls **Unit X-01**, a robot trapped inside a dangerous testing facility. Toxic barrels fall from above, and the objective is to survive as long as possible while avoiding collisions.

This project is being developed step by step to learn professional game development, JavaScript, AI concepts, and software engineering practices.

---

## Current Features

- Robot movement
- Boundary collision
- Falling toxic barrels
- Random barrel spawning
- Collision detection
- 5-life system
- Game Over screen

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- HTML Canvas
