## Prototype Version 0.1

### Gameplay
- Control Unit X-01 using the arrow keys.
- Avoid falling toxic barrels.
- Lose one life when hit.
- Start with 5 lives.
- The game ends when all lives are lost.

### Current Features
- Robot movement
- Boundary collision
- Falling obstacles
- Random spawning
- Collision detection
- Lives system
- Game Over
